# 💌 Покани ме на среща

Супер забавен сайт, с който пращаш на човек линк с покана за **романтична** или **приятелска** среща.
Поканеният вижда анимиран фон от сърца, умолително човече и въпроса „Искаш ли да излезеш с мен?“ —
с бутон **НЕ**, който бяга от мишката (и отскача при докосване на телефон 😈). След **ДА** избира
дата, час и какво да правите, а ти получаваш отговора му на специален линк.

Към поканата можеш да добавиш **своя снимка** и **гласово съобщение** (записано директно от
браузъра или качено като файл) — те се показват на поканения преди да отговори.

Сайтът е **двуезичен** (🇧🇬 Български / 🇬🇧 English) — бутонът горе вдясно сменя езика и изборът
се запомня. Всички текстове, закачки, дати и обобщения са преведени.

## Технологии
- **Frontend:** React + Vite + Tailwind CSS + Framer Motion
- **Backend:** Node + Express
- **База данни:** SQLite (better-sqlite3)

## Стартиране (за разработка)
```bash
npm install
npm run dev
```
- Frontend: http://localhost:5173
- API: http://localhost:5174 (Vite автоматично му препраща заявките към `/api`)

## Продукция
```bash
npm run build      # компилира фронтенда в /dist
npm start          # Express сервира /dist + API на един порт (PORT, по подразбиране 5174)
```

## Как работи
1. На началната страница попълваш име, вид покана (романтична/приятелска), пол на поканения и съобщение.
2. Получаваш **два линка**:
   - линк за **споделяне** (`/i/<id>`) — пращаш го на човека;
   - линк за **резултата** (`/r/<id>`) — запазваш го за себе си, тук виждаш отговора на живо.
3. Поканеният минава през стъпките и отговорът се записва в базата.

## Структура
```
src/                 React приложение
  components/         Сърца, анимирани герои, бягащ бутон „Не“, конфети
  pages/              Create / Invite / Results / 404
  api.js  lib.js      API клиент и помощни функции
server/               Express API + SQLite (data.db се създава автоматично)
  uploads/            Качени снимки и гласови съобщения (създава се автоматично)
```
